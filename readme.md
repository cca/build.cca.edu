# Build Lab Web Archive

This is a web archive of the Build Lab site which used to reside at https://build.cca.edu. It was created using [WebRecorder](https://webrecorder.net/)'s ArchiveWeb.page Chrome extension and the archive is displayed using their ReplayWeb.page tool.

The actual web archive file is `build.cca.edu.wacz`, the rest of this repo is just the HTML frame for replaying it.

**Note**: since the WACZ file is over 100mb we have not been able to get playback to work using GIitHub Pages. But the full file can be downloaded here or under [the **Releases** tab](https://github.com/cca/build.cca.edu/releases).
