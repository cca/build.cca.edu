# Studio Forward Web Archive

This is a web archive of the Studio Forward Public Programming site which used to reside at https://studioforward.cca.edu/public-programming/. It was created using [WebRecorder](https://webrecorder.net/)'s ArchiveWeb.page Chrome extension and the archive is displayed using their ReplayWeb.page tool.

The actual web archive file is `studio-forward.wacz`, the rest of this repo is just the HTML frame for replaying it.
